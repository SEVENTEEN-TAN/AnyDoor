安装指南

使用解压安装方式在浏览器中加载扩展：

Chrome

  1. 下载扩展压缩包并解压到本地目录。
  2. 在地址栏输入 chrome://extensions/ 打开扩展管理页。
  3. 开启右上角的“开发者模式”。
  4. 点击“加载已解压的扩展”，选择解压后的目录。

Edge

  1. 下载扩展压缩包并解压到本地目录。
  2. 在地址栏输入 edge://extensions/ 打开扩展管理页。
  3. 开启“开发者模式”。
  4. 点击“加载解压的扩展”，选择解压后的目录。

完成后，浏览器工具栏将出现 AnyDoor 图标。

使用指南

  1. 在需要保持登录状态的网站上，点击浏览器工具栏中的 AnyDoor 图标。
  2. 在扩展内选择“捕获会话”，上传当前站点的登录状态与相关令牌。
  3. 在另一台设备或浏览器中，按上述方式加载扩展后打开 AnyDoor。
  4. 在扩展内选择对应会话并点击“同步”，刷新页面即可恢复登录状态。

常见操作

  • 更新会话：在已登录页面再次执行“捕获会话”。
  • 删除会话：在扩展的会话列表中选择并删除不再需要的会话。
  • 暂停同步：在扩展设置中关闭同步开关即可临时停用。

快速开始（TL;DR）

  1. 安装扩展（商店或解压加载）。
  2. 在登录完成的站点点击扩展图标 → 捕获会话。
  3. 在另一设备安装扩展 → 选择会话 → 同步。

安全实践

  • 启用系统级登录保护（Windows Hello / Touch ID 等）。
  • 避免在不受信网络同步，优先使用受信环境。
  • 定期删除过期或不再使用的会话数据。

数据管理

  • 导出：在扩展设置中导出会话数据（如可用）。
  • 删除：在会话列表中选择会话并执行删除。
  • 撤回：在设置中关闭同步并清除本地缓存。

常见问题（FAQ）

  • 同步后仍需重新登录？ 部分站点启用设备绑定或多因子验证，需在目标设备完成额外验证流程。
  • 同步速度变慢？ 检查网络状况与扩展版本，建议在受信网络环境下使用并保持最新版本。
  • 是否支持团队共享？ 支持创建共享空间并设定权限，详见产品内“团队”功能（如您的套餐包含）。

故障排查

  • 确认浏览器未安装拦截类插件影响存储访问。
  • 在隐私设置中允许扩展访问站点数据（如被限制）。
  • 清理目标站点缓存后重试同步与恢复。
