手动安装步骤

Chrome

  1. 下载扩展压缩包并解压到本地目录。
  2. 在地址栏输入 chrome://extensions/ 打开扩展管理页。
  3. 开启右上角的“开发者模式”。
  4. 点击“加载已解压的扩展”，选择解压后的目录。

Edge

  1. 下载扩展压缩包并解压到本地目录。
  2. 在地址栏输入 edge://extensions/ 打开扩展管理页。
  3. 开启“开发者模式”。
  4. 点击“加载解压的扩展”，选择解压后的目录。
